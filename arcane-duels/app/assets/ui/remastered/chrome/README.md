# Arcane Duels — Remastered chrome kit

Modular illustrated UI assets derived from the user-approved ornamental concept.

- `header-player.png`: player plaque and portrait medallion; mirrored by CSS for the opponent.
- `status-medallion.png`: central phase/round plaque.
- `info-footer.png`: full-width card information frame with button wells.
- `spell-parchment.png`: spell-only frame used exclusively by the **Nuovo** card-art style.
- `sources/`: lossless chroma-key generation outputs retained for reproducibility.

The production PNG files were generated on a flat magenta key, converted to RGBA with soft matte/despill, and cropped to their non-transparent bounds. They are remastered assets, not original Astral Tournament resources.
